# Human Benchmark Visual Memory Practice - Level Selector

A browser extension that lets you practice the Visual Memory Test from humanbenchmark.com at any level with optional unlimited lives.

## Features

- **Custom Starting Level** — Skip to any level (1-50) for targeted practice
- **Unlimited Lives** — Disable the 3-life limit so you can keep practicing without restarting

## Installation

1. Download or clone this folder
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select this folder

## Usage

1. Go to [Human Benchmark Visual Memory Test](https://humanbenchmark.com/tests/memory)
2. Click the extension icon in your browser toolbar
3. Set your desired starting level
4. Check **Unlimited Lives** if you want to practice without the 3-life limit
5. Click **Save & Reload Page**
6. Click **Start** on the game

## Notes

- Settings are saved and persist across browser sessions
- The page must be reloaded for new settings to take effect
- Works only on the Visual Memory Test page

## Disclaimer

This extension is for personal practice only. It may stop working if Human Benchmark updates their website.
